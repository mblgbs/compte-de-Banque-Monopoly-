# Faux cartes bancaires Monopoly

Ce dépôt contient des **faux visuels de cartes bancaires** destinés à une partie de Monopoly maison.

## Contenu

- `cards/carte-bleue.svg`
- `cards/carte-rouge.svg`
- `cards/carte-verte.svg`
- `cards/carte-noire.svg`

Chaque carte est un fichier SVG modifiable (texte, nom, couleur, numéro).

## Personnalisation rapide

1. Ouvrir le fichier SVG voulu dans un éditeur texte ou un logiciel vectoriel (Inkscape, Figma, Illustrator).
2. Modifier :
   - le nom du joueur ;
   - le numéro de carte ;
   - la date d'expiration ;
   - le code banque.
3. Exporter en PNG si besoin pour impression.

## Note

Ces cartes sont prévues pour un jeu de société et ne sont pas des moyens de paiement.
