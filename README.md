# Compte de Banque Monopoly — Fausses CB

Ce mini-projet génère des **fausses cartes bancaires Monopoly** pour les joueurs.

## Fonctionnalités

- Génération d'une carte par joueur avec :
  - titulaire
  - numéro formaté (16 chiffres)
  - date d'expiration
  - CVV
  - banque émettrice fictive
- Interface web simple pour choisir le nombre de joueurs.
- Bouton pour régénérer un nouveau lot de cartes.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Choisir le nombre de joueurs (2 à 8).
3. Cliquer sur **Créer les CB Monopoly**.
4. Optionnel : cliquer sur **Régénérer** pour refaire des cartes.

## Notes

- Les données générées sont entièrement fictives et destinées au jeu.
- Aucun appel réseau ni stockage persistant.
